CSS Selectors, Attribute Selectors and Pseudo-elements/-classes

Example Name Description
* { } Universal selector applies to all elements in the document.
h1 { } Type selector matches element names
.box { } Class selector targets any elements whose class attribute has a value of box
p.box { } Chaining targets only <p> elements whose class attribute have a value of note
#unique { } ID selector matches the one element whose id attribute has the value of unique
li>a { } Child selector targets any <a> elements that are direct children of an <li> element, but no other <a> elements on the page
p a { } Descendant selector targets any <a> elements that sit inside a <p> element, even if there are other elements nested between them
h1+p { } Adjacent sibling selector (only supported from IE 7 on targets the first <p> element after any <h1> element, but no other <p> elements
h1~p { } General sibling selector (only supported from IE 7 on) targets all <p> elements after any <h1> element
p[class] { } Existence selector [ ] targets any <p> element with an attribute called class, whatever its value
p[class="dog"] { } Equality Selector [=] targets any <p> element with an attribute called class whose value is dog
p[class~="dog"] { } Space Selector [~=] targets any <p> element with an attribute called class whose value is a list of space-seperated words, one of which is dog
p[attr^"d"] { } Prefix Selector [^=] targets any <p> element with an attribute whose value begins with the letter "d"
p[attr*"do"] { } Substring Selector [=] targets any <p> element with an attribute whose value contains the letters "do"
p[attr$"g"] { } Suffix Selector [=] targets any <p> element with an attribute whose value ends with the letter "g"
p.intro:first-letter { } Pseudo-element acts like there is an extra element that can be addressed, in this case allows to style the first letter
p.intro:first-line { } Pseudo-element as before, in this case allows to style the first line of text
a:link { } Pseudo-class use the following 5 states in this order for links. This case allows styling for a link that has not been visited yet
a:visited { } Pseudo-class allows styling for a link that has been visited
a:hover { } Pseudo-class allows styling for when the user hovers over a link
a:active { } Pseudo-class allows styling for when an element is activated by the user (link is being clicked, or a button being pressed)
a:focus { } Pseudo-class allows styling for when the browser discoveres the user is ready to interact with an element (e.g. when the tab key is used to navigate through the interactive items on a page)

CSS clearfix

Description 

.clearfix:after {
    content: " ";
    visibility: hidden;
    display: block;
    height: 0;
    clear: both;
}
 to clear floating elements for the succeeding element (apply to parent element) - only necessary if not using flexbox and grid

HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing the actual table data.
<tr> Table Row The table row container.
<td> Table Data The table cell container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a <td> element should span.
rowspan Row Span Defines how many rows a <td> element should span.
